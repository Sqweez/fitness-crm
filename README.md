# Passera CRM

CRM-система для управления фитнес-клубами и сетями.

## Возможности

- **Управление клиентами** — база клиентов, сегментация, история посещений, интеграция с WhatsApp
- **Абонементы и услуги** — безлимиты, программы тренировок, солярий, заморозка, продление
- **Контроль посещений** — регистрация входа/выхода, выдача ключей, мониторинг зала
- **Финансы** — пополнения, продажи, выводы, раздельный учёт наличных и карт
- **Товары** — каталог бара и магазина, складской учёт по локациям
- **Персонал** — роли, права доступа, журнал действий сотрудников
- **Аналитика** — отчёты по продажам, клиентам, сотрудникам, графики
- **Мультиклуб** — управление сетью клубов из единой системы

Подробное описание функций: [FEATURES.md](./FEATURES.md)

## Технологии

- **Frontend:** Nuxt 2, Vue 2, Vuex, Vuetify 2, TailwindCSS
- **Backend:** Laravel (внешний API)
- **Авторизация:** JWT

## Запуск

Локальный запуск осуществляется через Docker.

```bash
# Запуск контейнера
docker-compose up --build

# Синхронизация node_modules после установки в контейнере
npm run sync:modules
```

Приложение будет доступно на `http://localhost:3000`

## Команды

```bash
npm run dev       # Запуск dev-сервера
npm run build     # Сборка для production
npm run start     # Запуск production-сервера
npm run generate  # Генерация статического сайта
```
